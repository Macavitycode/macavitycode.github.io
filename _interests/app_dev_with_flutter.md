I started working with flutter a while ago, and it has always been a useful
tool to know.
I have built quick apps for a number of use cases including lab mini projects
like [this one](https://github.com/Macavitycode/iiot_lab_mini_project), which
is a quick application to control an ESP32 over BlynkIoT framework.

The app is a simple screen consisting of two sliders which alter speeds of
different side wheels to enable track like motion.
