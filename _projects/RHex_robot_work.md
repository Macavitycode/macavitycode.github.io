The project is to realise a full RHex bot from 3D modelling to electronics and
algorithms.

The groundwork for the design, mechanical simulations and control system design
has been done through ROS by seniors.
I have the 3D model of the robot completed, and am working on the electronics.
I plan to have this work continued over a long time period as building
something from scratch like this takes a lot of time and effort.

Design considerations:
 - Minimize cost of manufacturing, 3D print if possible
 - Rugged bot with springy legs
 - Easy locomotion on uneven surfaces
 - Ground clearence
 - Use available electronics
 - Gait and motion planning algorithms\*
